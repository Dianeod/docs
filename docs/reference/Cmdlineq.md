`q` `-q` set quiet mode, i.e. no startup banner text or input prompts. Typically used where no console is required.

Syntax
------

    ~/q$ q -q

Example
-------

    ~/q$ q
    KDB+ 2.6 2010.01.14 Copyright (C) 1993-2010 Kx Systems
    ..
    q)2+2
    4
    q)

and with -q

    ~/q$ q -q
    2+2
    4
