`x` `mod` `y` returns the remainder of `x%y`. Applies to numeric types, and gives type error on sym, char and temporal types.

Syntax
------

    q)r:X mod Y

Example
-------

    q)-3 -2 -1 0 1 2 3 4 mod 3
    0 1 2 0 1 2 0 1
