The `exp` function computes e<sup>x</sup>, where e is the base of natural logarithms. The result is type float.

Syntax
------

    q)r:exp X

Example
-------

    q)exp 1
    2.718282
    q)exp 0.5
    1.648721
    q)exp -4.2 0 0.1 0n 0w
    0.01499558 1 1.105171 0n 0w
