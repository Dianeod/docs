The `sin` function computes the sine of its argument, taken to be in radians. The result is between `-1` and `1`, or null if the argument is null or infinity.

Syntax
------

    q)a:sin 0.5

Example
-------

    q)sin 0.5
    0.4794255
    q)sin 1%0
    0n
